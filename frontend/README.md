# CalmSense Dashboard

React + TypeScript dashboard for the CalmSense stress-detection project.
[Live demo →](https://urme-b.github.io/CalmSense/)

The trained Random Forest is exported to ONNX and runs **entirely in the browser**
(`onnxruntime-web`), so the prediction panel works on static hosting with no backend.
All reported numbers come from `src/results.json`, generated by the experiment scripts.

## Pages

- **Dashboard** — headline LOSO results and the within- vs subject-independent optimism gap
- **Signal Explorer** — real WESAD chest signals (baseline → stress → amusement)
- **Prediction** — runs the real model in-browser on uploaded or sample features
- **Explainability** — global mean |SHAP| feature importance and the motion-confound note
- **Model Comparison** — per-model accuracy, wrist-only, and cross-dataset results

## Develop

```bash
npm install
npm start          # http://localhost:3000
npm run build      # production build (also runs in CI deploy)
```

To regenerate the in-browser assets from a freshly trained model:

```bash
python scripts/export_onnx.py      # -> public/model.onnx + src/model_meta.json
python scripts/export_signals.py   # -> src/signals.json
```

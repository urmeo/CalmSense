# CalmSense Dashboard

React and TypeScript dashboard for the CalmSense stress-detection project. The trained
Random Forest is exported to ONNX and runs entirely in the browser via `onnxruntime-web`,
so prediction works on static hosting with no backend. All reported numbers come from
`src/results.json`, generated by the experiment scripts.

Live demo: https://urme-b.github.io/CalmSense/

## Pages

- Dashboard: headline LOSO results and the within- vs. subject-independent optimism gap
- Signal Explorer: real WESAD chest signals across baseline, stress, and amusement
- Prediction: runs the model in-browser on uploaded or sample features
- Explainability: global mean absolute SHAP feature importance and the motion-confound note
- Model Comparison: per-model accuracy, wrist-only, and cross-dataset results

## Development

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build (also run in CI deploy)
```

## Regenerating in-browser assets

Run these from the repository root after retraining the model:

```bash
python scripts/export_onnx.py      # public/model.onnx and src/model_meta.json
python scripts/export_signals.py   # src/signals.json
```
